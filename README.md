## Basic GO CLI

1. Run the code:

    ```
    go run <app-name>.go
    ```

2. Run the main code, if module exists:

    ```
    go run .
    ```

3. Build module:

    ```
    go mod init example.com/<app-name>
    ```

4. Build executable:

    ```
    go build
    ```

5. Run executable:
    ```
    ./<app-name>.exe
    ```
